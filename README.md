# Pig posture recognition Kaggle challenge

Link: https://www.kaggle.com/competitions/pig-posture-recognition/data

## Challenge description

Pig posture recognition is a key task in Precision Livestock Farming, enabling automated monitoring of animal health and welfare. In this competition, participants develop computer vision models to classify pig postures from 2D images captured under realistic farm conditions, including occlusion, lighting variation, and multiple camera views. Using the provided bounding boxes, participants must predict the posture of each pig instance and submit their results in a standardized submission CSV file.
